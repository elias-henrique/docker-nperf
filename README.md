# Docker Nperf
> nPerf determina com precisão a capacidade da sua conexão internet.
## Instalação
### Requeridos
  * `docker`
  * `docker-composer`

## How to run
1. Baixe `git clone https://github.com/elias-henrique/nperf.git`
2. Entre com `cd nperf/`
3. Baixe e inicie o container com `docker-compose up -d`
